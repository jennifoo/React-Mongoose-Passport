# React-Mongoose-Passport

Application with Authentication with a React Front-end and an Express/Mongoose Back-end.

## Installation

### Clone

```bash
git clone https://github.com/viet-vo/React-Mongoose-Passport.git
cd React-Mongoose-Passport
npm install
npm start
```
